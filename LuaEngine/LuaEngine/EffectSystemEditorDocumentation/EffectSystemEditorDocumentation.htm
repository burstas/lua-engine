Effect System Editor Documentation

Preperation

1, Open the file specified in hge.ini with the name of ResourceFile under section [Resource].

2, Type your texture file name with the name of Type_X under section [Texture]. X is a number start from 1. Remember editor uses X-1 as texture number.

3, Check EffectsysFolder under section [Folder] is your working folder.

4, Check Resource/EditorFont/Font.fnt and Font.png exists.

Status Bar

（X，Y，Z）nLives=particles alive    average FPS per 35 frames

Preview

Panel

The first line

Panel select tabs

Panels marked with EB (basicinfo), EE (emitterinfo) and EA (affectorinfo) are three different control levels of Effect System.

Shotcut keys

When Input is not highlighted: Left Arrow, Right Arrow and Tab.

The Second Line

Title and Add/Delete button.

Title

Under EB

Filename.

Under EE

Emitter_FilenameNumber_emitterID.

Under EA

AffectorType_FilenameNumber_emitterID_affectorID.

Add/Delete button

Add

Under EB

Add EffectSystem File.

UNDER EE

Add an emitter of current EffectSystem.

UNDER EA

Add an affector of current emitter.

Delete (When Right Ctrl key is pressed)

UNDER EB

Delete this EffectSystem.

UNDER EE

Delete this emitter.

UNDER EA

Delete this affector.

Shotcut keys

End or Right Ctrl + End. Up Arrow and Down Arrow when panel is empty.

Parameter panel

You can type parameter in Input box and Enter key for confirm.

The second line from the bottom

Input Box

You can type in Input box only while the box is activated.

PageUp、PageDown

UNDER EB

Previous/Next EffectSystem.

UNDER EE

Previous/Next emitter.

UNDER EA

Previous/Next affector.

Shotcut keys

         PageUp and PageDown without pressing Right Ctrl.
